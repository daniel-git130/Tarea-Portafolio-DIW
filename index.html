Sobre mi

Soy estudiante de desarrollo web y actualmente me estoy formando en tecnologías tanto de frontend como de backend. Trabajo con HTML, CSS y Bootstrap para el diseño y maquetación de interfaces web, y con JavaScript y PHP para añadir funcionalidad y lógica a las aplicaciones. Además, estoy adquiriendo conocimientos en AWS y en el despliegue de aplicaciones web utilizando máquinas virtuales, lo que me permite comprender el proceso completo desde el desarrollo hasta la puesta en producción. Mi objetivo es seguir mejorando mis habilidades técnicas y ganar experiencia práctica en el desarrollo y despliegue de aplicaciones web.
